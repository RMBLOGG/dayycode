Syarat & Ketentuan

Terakhir diperbarui: Januari 2025

1. Ketentuan Umum

Dengan melakukan pembelian di Dayy Code, kamu dianggap telah membaca, memahami, dan menyetujui seluruh syarat dan ketentuan yang berlaku. Dayy Code berhak mengubah syarat dan ketentuan ini sewaktu-waktu.

2. Produk & Lisensi

  • Semua produk yang dijual adalah source code digital berupa file yang dapat diunduh.
  • Pembelian memberikan hak lisensi personal/komersial untuk menggunakan source code pada proyek milik pembeli.
  • Dilarang menjual kembali, mendistribusikan, atau mengklaim source code sebagai karya sendiri.
  • Pembeli diperbolehkan memodifikasi source code sesuai kebutuhan.

3. Pembelian & Pembayaran

  • Pembayaran dilakukan secara manual melalui metode transfer yang tersedia.
  • Pesanan akan diproses setelah bukti pembayaran diterima dan dikonfirmasi.
  • Harga yang tertera adalah harga final dalam Rupiah (IDR).
  • Dayy Code berhak membatalkan pesanan yang terbukti melakukan kecurangan.

4. Pengiriman & Download

  • Setelah pembayaran dikonfirmasi, link download akan dikirimkan melalui email.
  • Link download memiliki batas waktu dan jumlah unduhan tertentu.
  • Jika link download bermasalah, segera hubungi kami via WhatsApp.

5. Kebijakan Refund

Karena produk yang dijual berupa file digital, kami tidak menerima refund setelah file berhasil diunduh. Pastikan kamu membaca deskripsi produk dengan cermat sebelum melakukan pembelian. Jika terjadi masalah teknis pada produk, kami akan berusaha membantu melalui WhatsApp.

6. Batasan Tanggung Jawab

Dayy Code tidak bertanggung jawab atas kerugian yang timbul akibat penggunaan produk yang tidak sesuai dengan petunjuk, atau akibat modifikasi yang dilakukan pembeli. Dukungan teknis yang kami berikan bersifat best-effort dan tidak mengikat.
